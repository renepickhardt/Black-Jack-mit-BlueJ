Dies ist eine (absichtlich) unfertige und unvollständige Sammlung von Klassen, 
die das Spiel BlackJack von Computern ermöglichen soll. 

Das Projekt dient dem Zweck, dass Schülerinnen und Schüler mit minimalen 
Programmiererfahrungen und theoretischem Wissen eigenständig ihre Kenntnisse erweitern
können. 

Folgende praktische Erfahrungen werden voraus gesetzt:
* Kontrollstrukturen: if, else, else if
* for schleifen
* Deklaration und Implementierung von Methoden

Folgende theoretische Erfahrungen werden vorausgesetzt:
* Vorstellung des Objektbegriffes insbesondere:
** Attribut
** Methode

Folgendes Wissen soll verfestigt und vertieft werden:
* Sichtbarkeit von Variablen, Objekten und Methoden
* Umgang mit Arrays (am Beispiel der ArrayList Klasse aus dem Paket java.util)
* Scope von Variablen und Objekten
* Vererbung
* API Design
* Programmabläufe / Datenfluss
* Kreativer Umgang mit einer Programmiersprache
* Die Fähigkeit femden Code zu lesen und zu verstehen
* Strukturiertes Umsetzen eines festen Regelwerkes in ein automatisiertes Computerprogramm

Folgendes Wissen kann optional verfestigt und vertieft werden:
* Design Pattern (Insbesondere Observer)
* API Design
* Netzwerkprogrammierung (Für ein Turnier)
* Grundlagen von Simulationen, Statistik, Maschinellem Lernen
